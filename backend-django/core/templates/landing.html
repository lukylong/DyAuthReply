使用指南

从安装到开始自动回复的完整流程。

① 安装桌面客户端

macOS：

  1. 下载 .dmg，打开后把 DyAuthReply 拖入「应用程序」。
  2. 由于安装包未经 Apple 公证，首次打开前在「终端」执行解除隔离：
/usr/bin/xattr -cr "/Applications/DyAuthReply.app"
  1. 从「启动台」打开；若仍被拦截：系统设置 → 隐私与安全性 → 仍要打开。

Windows：

  1. 运行 *-setup.exe 完成安装。
  2. SmartScreen 提示未知发布者时，点「更多信息 → 仍要运行」。
  3. 首次启动会初始化数据库，启动页可能等待 30–60 秒。

② 安装浏览器登录态插件

  1. 下载插件 .zip 并解压到一个固定目录。
  2. 打开 chrome://extensions（Edge 为 edge://extensions）。
  3. 打开右上角「开发者模式」，点「加载已解压的扩展程序」，选择解压出的 douyin-cred-extractor 文件夹。
  4. 建议把「抖音登录态提取器」图标固定到工具栏。
为什么用插件而不是控制台脚本：抖音的 sessionid 是 HttpOnly Cookie，网页 JS 读不到，只有插件的 chrome.cookies 权限能拿到完整 Cookie。插件不联网、不上传任何数据。

③ 采集登录态（生成导入串）

  1. 浏览器登录 抖音创作者中心，进入私信管理页：
    https://creator.douyin.com/creator-micro/data/following/chat
  2. 在该页面点插件图标，弹窗会自动抓取 Cookie / web_protect / keys 并生成 一键导入串（DYCRED1. 开头）。
  3. 点「复制一键导入串」。
切换账号后必须先刷新页面再抓取，否则会出现「新 Cookie + 旧账号信息」的混乱数据，插件会黄色警告并拦截。

④ 在客户端导入并托管

  1. 打开 D助手 客户端，进入「抖音账号」页，点「导入抖音号」。
  2. 把一键导入串粘贴到输入框，确认导入；校验通过后账号即上线。
  3. 含 web_protect + keys 时为「可发送」，否则为「仅接收」。
  4. 在「回复规则 / 模板」配置关键词与回复内容，开启账号的「自动回复」开关并设置日限额。
  5. 在「私信工作台」可实时查看会话、手动回复。

⑤ 常见问题

  • 掉线 / 凭证过期：重新在创作者中心登录，用插件再抓一次，到客户端点「重新导入」覆盖即可。
  • 多账号托管：单机建议不超过 10 个账号；账号过多会增加内存占用、回复延迟与风控关联风险，建议分散到多台设备。
  • 数据与日志：macOS 在 ~/Library/Application Support/DyAuthReply/，Windows 在 %APPDATA%\DyAuthReply\。
